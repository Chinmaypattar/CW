# Timer App Assignment

A simple, interactive timer application built with React. This app allows users to add, edit, manage, and run multiple timers simultaneously. Each timer has the ability to start, stop, restart, and display its remaining time with a countdown.

## Features

- **Multiple Timers**: Run as many timers as you need simultaneously.
- **Start/Stop Timers**: Toggle the timer between running and stopped states.
- **Restart Timers**: Reset the timer to its original duration.
- **Timer Notifications**: Receive a notification when a timer finishes.
- **Edit Timers**: Update the title, description, and duration of each timer.
- **Audio Alerts**: Play an audio alert when a timer finishes.
- **Persistent Timers**: Timers are stored in local storage to persist across page refreshes.
- **Responsive Design**: The app works well on all screen sizes, providing a great user experience on desktop and mobile.

## Tech Stack

- **React**: The frontend framework for building the app UI.
- **Redux Toolkit**: For state management, handling timers and actions.
- **Sonner**: For notifications when a timer finishes.
- **Lucide Icons**: For icons used throughout the app.
- **TypeScript**: Type safety for better development experience.
- **Tailwind CSS**: Utility-first CSS for styling.

## Installation

To get started with the project, follow these steps:



## **Project Setup**

1. Clone the repository:  
   ```bash
   git clone https://github.com/CW-Codewalnut/timer.git
   ```

2. Install dependencies:  
   ```bash
   npm install
   ```

3. Start the development server:  
   ```bash
   npm run dev
   ```

4. Run tests:  
   ```bash
   npm vitest
   ```
